# Firework

## Installation _(from source)_

Common

```bash
# bash, powershell:
pip3 install pygame==2.0.0.dev6 numpy
# or:
pip3 install -r requirements.txt
```

## Getting Started

### Running

Run game through command line with following options:

```bash

# using python interpreter directly:
python3 firework

# or the Makefile:
make run
```
